# SandboxSystem2D

<!-- # Short Description -->

Comprehensive block placement and removal capabilities in a 3D overlooking sandbox.   
Blocks can be placed only at the points set in the grid.

# Advantages

Provides powerful functionality for creating runtimes for the following look-down sandboxes.   
- Provide a Scriptableobject that changes the shape of the block and sets the ID
- Place blocks
- Remove blocks
- Replace blocks
- Block placement preview
- Indestructible blocks
- Automatic shape change when identical blocks are lined up
- Faster processing by reducing block surface depiction
- Installation and deletion of multiple blocks by drag motion
- Issue a block ID list to save the map
